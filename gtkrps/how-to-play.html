How to Play GTKRPS

GTKRPS (Growtopia Korea Private Server) adalah server privat untuk game Growtopia. Ikuti panduan berikut untuk menghubungkan perangkatmu.

üì± Android

  1. Download Powertunnel APP here.
  2. Click on "Powertunnel" button to copy the host link.
  3. Open Powertunnel app ‚Üí three-line icon ‚Üí host settings icon.
  4. Paste the copied host URL and set update period to "on start".
  5. Now log in to your GTPS!

üçé iOS

  1. Download "Surge 5" from App Store.
  2. Press "OK".
  3. Click "Default.conf".
  4. Click "IMPORT" - download profile from URL.
  5. Paste host URL, click "OK" ‚Üí "Done".
  6. Press "SETUP" ‚Üí agree to policy by clicking "OK" and "Allow".
  7. Open Growtopia and connect!

üíª Windows

  1. Press ‚äû Win+R to open "Run".
  2. Type C:\Windows\System32\drivers\etc.
  3. Find file "hosts".
  4. Right-click ‚Üí Edit with Notepad/text editor.
  5. Add the IP from GTKRPS server.
  6. Save file and log on to Growtopia client.
