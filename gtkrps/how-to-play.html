How to Play GTKRPS

GTKRPS (Growtopia Korea Private Server) adalah server privat untuk game Growtopia. Ikuti panduan berikut untuk menghubungkan perangkatmu.

üîπ 1. Character Setup

  • Buat karakter lengkap: nama, umur, kepribadian, dan latar belakang.
  • Pilih role yang tersedia (Hero, Villain, Support, Specialist).
  • Gunakan format karakter sesuai template komunitas.

üîπ 2. Basic Gameplay

  • Pemain berinteraksi melalui teks roleplay.
  • Gunakan aksi bergiliran sesuai turn order.
  • Jangan melakukan auto-win atau auto-hit.
  • Aksi besar (power move / special) harus mengikuti cooldown.

üîπ 3. Combat Rules

  • Gunakan *dice roll* bila diinstruksikan.
  • Serangan harus bisa di-counter oleh pemain lain.
  • Damage mengikuti sistem level yang berlaku.

üîπ 4. Story & Events

  • Ikuti timeline dari admin atau event master.
  • Karakter boleh ikut arc besar atau mission side-story.
  • Dilarang merusak alur cerita major tanpa izin.

üîπ 5. Behavior Rules

  • Hormati semua pemain.
  • Dilarang toxic, NSFW, atau godmodding.
  • Gunakan bahasa sesuai channel roleplay.

üîπ 6. Leveling System

  • Naik level dengan menyelesaikan misi.
  • EXP diperoleh dari partisipasi aktif.
  • Level tinggi membuka akses special power.

üì± Android

  1. First, download Powertunnel APP from here.
  2. Click on the "Powertunnel" button to copy the host link.
  3. Open the Powertunnel app, click the three-line icon, then host settings icon.
  4. Paste the copied Powertunnel host URL and set host update period to "on start".
  5. Now you can log in to your GTPS!

üçé iOS

  1. Download app called "Surge 5" from the App Store.
  2. Press "OK".
  3. Click on "Default.conf".
  4. Click on "IMPORT" - download profile from URL.
  5. Paste the host URL copied from above, then click "OK" and "Done".
  6. Press "SETUP" and agree to the policy by clicking "OK" and "Allow" for VPN configuration.
  7. Open "Growtopia" and connect!

üíª Windows

  1. Press ‚äû Win+R (Windows Key + R) to open the "Run" window.
  2. Type C:\Windows\System32\drivers\etc in the textbox.
  3. Find the file named hosts.
  4. Right-click the file and edit it with Notepad or any text editor.
  5. Add the IP from GTKRPS server.
  6. Save the file and log on to your Growtopia client.

Ikuti semua langkah di atas agar bisa terhubung dengan GTKRPS dengan aman dan lancar!
